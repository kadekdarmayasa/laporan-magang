    3. Quality Control adalah proses pengecekan yang dilakukan oleh bisnis atau perusahaan, untuk memastikan kualitas produk sesuai standar yang ada. Berikut beberapa manfaat dari Quality Control :

    1. Produksi Lebih konsisten.
    2. Meningkatkan Efisiensi.
    3. Menjaga Kepuasan Pelanggan.
    4. Memungkinkan Bisnis Untuk Berkembang.

    Quality Control yang dilakukan oleh PT. Bamboomedia adalah pada aplikasi SpeedQ. Aplikasi yang menjadi salah produk yang dimiliki oleh PT. Bamboomedia. Quality Control yang ditugaskan oleh pihak PT. Bamboomedia adalah pada bagian penginputan data counter. Berikut adalah langkah – langkahnya :

    1. Masuk pada aplikasi SpeedQ.
    2. Jika sudah memiliki akun silahkan melakukan sign in sebagai office, jika belum silahkan sign up terlebih duhulu, dan buat jumlah counter sebanyak 10, yang memiliki operasional, additional, dan off day berbeda – beda pada masing – masing counter.
    3. Jika sudah selesai membuat counter, silahkan log out, dan hapus cache pada aplikasinya.
    4. Masuk kembali pada aplikasi SpeedQ, dan sign in sebagai counter yang sudah dibuat.
    5. Setelah sign in, lakukan input data sebanyak 40 sampai 60 antrian.
    6. Jika sudah selesai, silahkan log out kembali, dan hapus cache pada aplikasinya.
  1. Installasi Dan Konfigurasi Flutter

    2. STEP 1 : Installasi Flutter

    1. Buka situs resmi dari flutter pada link berikut ini Situs Resmi Flutter
    2. Jika sudah maka tampilannya akan seperti di bawah ini.
    3. Silahkan pilih sesuai dengan OS masing - masing.
    4. Scroll lalu cari flutter SDK dengan nama folder flutter_windows_2.5.3-stable.zip
    5. Jika sudah selesai melakukan installasi, sekarang ekstrak file tersebut dan letakkan dalam direktori yang di inginkan. Jangan letakkan file tersebut dalam direktori seperti C:\Program Files\, itu memerlukan hak istimewa yang lebih tinggi. Sebagai Contoh; C:\Users\Kadek-Darmayasa\flutter
    6. Sekarang tambahkan flutter pada PATH, caranya adalah dari bilah pencarian pada taskbar di windows, masukkan env dan pilih menu Edit the system menu environment variables, jika sudah pergi ke menu advanced > Environment Variables, sekarang lihat pada System Variables apakah sudah ada entri bernama Path, jika sudah, tambahkan path lengkap dari “flutter/bin”. Gunakan ; untuk tanda pemisah dari nilai yang ada. Jika entri tidak ada, buat variabel baru, dan beri nama Path dengan path lengkap dari “flutter/bin” sebagai nilainya.
    7. Untuk memastikan bahwa flutter sudah terinstall, pergi ke command prompt, dan ketik perintah berikut ini flutter –version.

      8. ini menandakan bahwa flutter sudah berhasil terinstall.

    STEP 2 : Download Android Studio

    1. Unduh dan Install Android Studio pada situs Situs Resmi Android Studio
    2. Jika Sudah terinstall, buka aplikasinya, dan buka panduan Android Studio Setup Wizard. Ini menginstall Android SDK, Android SDK Command-line Tools, dan Android SDK Build-Tools Terbaru, yang diperlukan oleh flutter saat mengembangkan aplikasi untuk Android.

    STEP 3 : Siapkan Perangkat Android

    1. Terdapat 2 cara jika ingin menjalankan aplikasi flutter, yaitu dengan emulator dan perangkat Android pribadi. Disini saya meggunakan perangkat Android pribadi.
    2. Untuk menjalankan dan menguji Flutter di perangkat Android, Kita memerlukan perangkat Android yang menjalankan Android 4.1 (API level 16) atau lebih tinggi.
    3. Jika sudah memenuhi persyaratan di atas. Ikuti langkah langkah berikut ini untuk menjalankan Flutter di perangkat Android.
      1. Aktifkan Developer Options dan USB Debugging pada perangkat Android.
      2. Gunakan kabel USB, untuk menguhubungkan perangkat Android dan laptop.
      3. Pada Command Prompt jalankan perintah flutter devices untuk memverifikasi bahwa flutter mengenali perangkat android yang terhubung.
      4. Disini perangkat androidnya, sudah berhasil terhubung yaitu vivo 1938 (mobile).

    STEP 4 : Setuju Dengan Lisensi Android

    Sebelum dapat menggunakan Flutter, Kita harus menyetujui lisensi platform Android SDK.

    1. Pastikan sudah memiliki versi Java 8 yang terinstall dan JAVA_HOME pada environment variables disetel ke folder JDK.
    2. Jika belum memiliki Java yang terinstall pada laptop, silahkan kunjungi situs berikut untuk installasinya Java JDK.
    3. Silahkan pilih file yang bernama x64 Installer, lalu download. Setelah di download ekstrak file tersebut dan letakkan pada direktori yang diinginkan. Sebagai contoh; C:\Progam Files\Java
    4. Tambahkan pada Path folder java jdk yang terinstall, caranya sama seperti menambahkan path flutter. Tambahkan varibel baru pada System Variables, dengan nama JAVA_HOME, dan nilainya diisi dengan path lengkap dari “Java/jdk-17”.
    5. Buka Command Prompt, dan jalankan perintah flutter doctor –android-licenses untuk menerima lisensi.
    6. Setujui semua persyaratan setiap lisensi.
    7. Setelah selesai menyetujui lisensi jalankan flutter doctor, untuk mengkonfirmasi bahwa Kita sudah siap menggunakan flutter dan untuk memastikan keperluan lain yang perlu diinstall.

      8. Disini kekurangannya adalah Visual Studio yang tidak terinstall pada perangkat komputer. Tetapi itu tidak berpengaruh terhadap pengembangan aplikasi menggunakan flutter. Sebagai alternatif saya menggunakan Visual Studio Code untuk menggantikan Visual Studio.

    8. Untuk install Visual Studio Code Silahkan kunjungi link berikut ini Visual Studio Code
    9. Intallasi Ekstensi Flutter dan Dart di Visual Studio Code

      10. Pergi ke menu File>Preferences>Extensions, lalu pada bilah pencarian ketik Dart dan Install. Untuk install Flutter juga sama, pada bilah pencarian ketik Flutter dan Install. Sekarang Visual Studio Code siap untuk menjalankan file flutter dan dart.
